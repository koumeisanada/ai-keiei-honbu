# ケイタ式 フォルダ

## ビジネスモデル
国内仕入・海外輸出販売

## フォルダ構成
・資料/：ケイタ式の講義資料・設計図を保存
・AI自動化分析/：各工程のAI活用可能性の分析結果
・自動化スクリプト/：実際の自動化スクリプト

## 使い方
1. 資料/フォルダに講義資料を保存する
2. Claude Codeに「ケイタ式の資料を分析して自動化できる部分を教えて」と入力する
3. AI自動化分析/フォルダに分析結果が保存される

## 分析依頼プロンプト
資料を保存した後に以下をClaude Codeに入力：

「Desktop/AI経営本部/物販ビジネス/ケイタ式/資料/
の全ファイルを読み込んで
ケイタ式のビジネスモデルの各工程を分析し
AIと自動化で効率化できる部分を
優先度付きで全てリストアップしてください。
分析結果はDesktop/AI経営本部/物販ビジネス/ケイタ式/AI自動化分析/
に保存してください。」
