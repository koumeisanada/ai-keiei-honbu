# SKILL：講義資料・マニュアル作成（Word形式）

## スキル概要

このスキルは、AI経営本部の講義資料・マニュアルをWord形式で作成するための完全な手順です。
デザインシステムに完全準拠した、プロフェッショナルな資料を自動生成します。

**成功実績：**
- AI経営本部_完全構築マニュアル_Ver1.2（2026年4月）
- 11章構成、テーブル多数、完全なデザイン適用

## トリガー条件

以下のキーワードが含まれる場合、このスキルを使用：
- 「講義資料を作成して」
- 「マニュアルを作成して」
- 「Word形式で資料を作って」
- 「〇〇のマニュアルを完成させて」

## デザインシステム（必ず遵守）

### 基本設定
- **ツール**: docx-js（npm: docx@9.5.3）/ Node.js
- **ページ**: A4（11906 × 16838 DXA）
- **マージン**: 1080 DXA（全方向）
- **フォント**: Meiryo 統一
- **出力**: 必ず.docxファイルで出力

### カラーパレット
```javascript
const COLORS = {
  NAVY: '1A1A2E',        // 見出しバー背景・タイトル文字
  PURPLE: '534AB7',      // セクション区切り・アクセント
  LIGHT_PURPLE: 'EDE9F6', // テーブル偶数行背景
  WHITE: 'FFFFFF',        // バー内文字・テーブルヘッダー文字
  GRAY: 'F5F5F5',         // テーブル奇数行背景
  DARK: '222222',         // 本文テキスト
  ORANGE: 'FF6B35'        // h3のアクセント
};
```

### フォントサイズ一覧

| 要素 | ポイント | docx size値 | 用途 |
|---|---|---|---|
| 表紙タイトル | 28pt | 56 | マニュアルタイトル |
| 表紙サブタイトル | 20pt | 40 | バージョン・日付 |
| h1（章タイトル） | 14pt | 28 | 第1章、第2章など |
| h2（セクション見出し） | 12pt | 24 | ◆ で始まる見出し |
| h3（小見出し） | 10pt | 20 | ▶ で始まる見出し |
| 本文 | 9.5pt | 19 | 通常テキスト |
| コードブロック | 9pt | 18 | コード |

**重要：** 講義資料は22pt本文だが、マニュアルは9.5pt

**h1（章タイトル）：**
- 色：NAVY
- サイズ：28（14pt）
- Bold：true
- 上マージン：800 / 下マージン：240

**h2（セクション見出し）：**
- 前置：「◆ 」（PURPLE）
- 色：NAVY
- サイズ：24（12pt）
- Bold：true
- 上マージン：400 / 下マージン：160

**h3（小見出し）：**
- 前置：「▶ 」（ORANGE）
- 色：PURPLE
- サイズ：20（10pt）
- Bold：true
- 上マージン：320 / 下マージン：120

### テーブルデザイン

**ヘッダー行：**
- 背景：NAVY / 文字色：WHITE / Bold：true / サイズ：19 / 配置：中央揃え

**データ行：**
- 奇数行背景：WHITE / 偶数行背景：LIGHT_PURPLE / 文字色：DARK / サイズ：19

**セルマージン：** top/bottom: 80 / left/right: 140

**テーブル幅：** 9746 DXA（WidthType.DXA使用）
- **PERCENTAGE禁止**（レイアウト崩れの原因）

## 作成プロセス（2ステップ）

### Step 1: Markdown形式で内容を作成

まず、内容をMarkdown形式で作成します。

### Step 2: docx-jsスクリプトでWord形式に変換

Markdown → Word変換スクリプトを作成して実行します。

## チェックリスト

- [ ] 箇条書きが「・」（Unicodeの•ではない）

## トラブルシューティング

### 問題1: テーブルレイアウトが崩れる
**原因:** WidthType.PERCENTAGEを使用している
**解決策:** `width: { size: 9746, type: WidthType.DXA }`

### 問題2: フォントサイズが意図と異なる
**原因:** size値の計算ミス
**解決策:** ポイント × 2 = size値（9.5pt = size: 19）

### 問題3: 本文色が#333333になっている
**原因:** 講義資料用の設定を使用している
**解決策:** マニュアルは#222222（DARK）を使用
