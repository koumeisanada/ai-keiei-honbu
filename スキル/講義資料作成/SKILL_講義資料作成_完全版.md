# SKILL：講義資料・マニュアル作成（Word形式）

## スキル概要
このスキルは、AI経営本部の講義資料・マニュアルをWord形式で作成するための完全な手順です。
デザインシステムに完全準拠した、プロフェッショナルな資料を自動生成します。

**成功実績：**

- 米国テック大手8社_最新動向レポート（2026年4月）

## デザインシステム（必ず遵守）

### 基本設定
- **ツール**: docx-js（npm: docx@9.5.3）/ Node.js
- **ページ**: A4（11906 × 16838 DXA）

- **出力**: 必ず.docxファイルで出力

## トラブルシューティング

### 問題1: テーブルレイアウトが崩れる
**原因:** WidthType.PERCENTAGEを使用している
**解決策:** `width: { size: 9746, type: WidthType.DXA }`

### 問題2: フォントサイズが意図と異なる
**原因:** size値の計算ミス
**解決策:** ポイント × 2 = size値（9.5pt = size: 19）

### 問題3: shading validation error
**原因:** w:shd要素にval属性がない
**解決策:** XMLを unpack → `sed -i 's/<w:shd w:fill="1A1A2E"\/>/<w:shd w:fill="1A1A2E" w:val="solid"\/>/g'` → pack

### 問題4: 本文色が間違っている
**原因:** 講義資料とマニュアルで本文色が異なる
**解決策:**
- 講義資料: #333333（BODY_COLOR）
- マニュアル: #222222（DARK）
