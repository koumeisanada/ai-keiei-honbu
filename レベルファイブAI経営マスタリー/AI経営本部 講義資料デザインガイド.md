# AI経営本部 講義資料デザインガイド
- 別のAIマネージャー・本部AIへの引き継ぎ文書
- 📋 このドキュメントについて
	このドキュメントは、「レベルファイブ AI経営マスタリー」の講義資料を作成する際の、**完全なデザイン仕様**を記載しています。
	別のAIマネージャーや本部AIが講義資料を作成する際は、このガイドに従ってください。
- 🎨 カラーパレット（完全準拠）
	**重要**: 本文の色は #222222 ではなく **#333333** です。
- 📏 フォントサイズ（完全準拠）
	- 表紙
	- 本文
	- 見出し
	- テーブル
		**重要原則**: 表紙のタイトル以外は、**全て9.5〜10.5pt**に収める。大きすぎるとアホっぽく見える。
- 🔤 フォント
	- 基本フォント
		**注**: 元の資料では全てメイリオで統一されています。
	- ビジネス標準フォント（参考）
		もし将来的にフォントを変更する場合は、以下を参考にしてください：
		- **本文（長文・正式書類）**: 游明朝, MS明朝
		- **見出し・プレゼン・強調**: 游ゴシック, メイリオ, BIZ UDゴシック
		- **英文**: Arial, Calibri, Times New Roman
- 📐 スペーシング（段落間隔）
	**重要**: スペーシング値は小さめに。80や320は大きすぎる。
- 📊 テーブルデザイン
	- 基本構造
	- セル内マージン
	- ボーダー
	- テーブル幅
		**重要**: 幅はDXA指定。PERCENTAGE禁止。
- 📝 見出しデザイン
	- h1（章タイトル）
	- h2（セクション見出し）
	- h3（小見出し）
		**重要**: 見出しマーク（◆、▶）は不要。シンプルにテキストのみ。
- 📌 箇条書き
	- 基本スタイル
	- 例
		**重要**: 箇条書きも9.5ptに統一。大きすぎない。
- 🚫 禁止事項
	- 1. 「WHY」という文字の使用禁止
		「WHY」は真田孔明の個人的な裏設定であり、受講生向け資料には記載しない。
	- 2. 大きすぎる文字サイズ禁止
	- 3. 過度なスペーシング禁止
	- 4. 色の間違い禁止
- 🎯 ページ設定
- 📂 ファイル出力ルール
	- ファイル形式
	- 保存先（大原則）
		**重要**: GitHubにpushされて初めて「資産」。Desktop・GitHub変更禁止・大原則。
		- [ ] 全ての文字サイズが9.5〜10.5pt（表紙タイトル以外）
		- [ ] 本文の色が #333333
		- [ ] 「WHY」という文字が含まれていない
		- [ ] テーブルヘッダーがNAVY背景、白文字
		- [ ] テーブル奇数行がGRAY、偶数行がLIGHT_PURPLE
		- [ ] 見出しバー（h1）がNAVY背景、白文字
		- [ ] 箇条書きが9.5pt
		- [ ] .docx、.md、.txtの3形式で保存
		- [ ] ~/Desktop/AI経営本部/... に保存
		- [ ] GitHubにpush完了
