# Claude Code 作業ルール

## マスター原稿の取り扱い

### 絶対禁止事項

**本文/フォルダ内の既存ファイルを変更・削除すること**

- .txtファイル：孔明マスター原稿（読み取り専用）
- .mdファイル：一切触らない

孔明の明示的指示がない限り、既存ファイルに触らないこと。

### 許可される作業

- ファイルの閲覧・参照
- 新規ファイルの作成（孔明の指示後のみ）
- Git操作（add/commit/push）

### マスター原則

```
本文/フォルダ内の.txtファイル = 孔明が直接編集したマスター原稿
```

AIは読み取り専用として扱う。

### 新規ファイル作成時の注意

- QUEST②など新規コンテンツ作成時は、必ず孔明の指示を待つ
- 作成後は.md + .txtの2形式で保存
- GitHub pushまで実行

### Git管理

孔明がファイル編集後、以下のコマンドでpush：

```bash
cd ~/Desktop/AI経営本部/レベルファイブAI経営マスタリー/
git add LINEステップメール/QUEST1/本文/
git commit -m "QUEST1原稿：孔明マスター修正反映"
git push origin main
```
