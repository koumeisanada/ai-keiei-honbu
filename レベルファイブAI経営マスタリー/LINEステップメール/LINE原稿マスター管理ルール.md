# LINE原稿マスター管理ルール

## 絶対原則
**本文/フォルダ内の全.txtファイルは「孔明マスター原稿」である**

### .txtファイル = 孔明が直接編集するマスター原稿

### .mdファイル = 一切触らない（参照のみ）

## AI作業ルール

### 禁止事項
- 既存.txtファイルの変更・削除（孔明の明示的指示がある場合のみ実行）
- .mdファイルへの変更・削除・作成

### 許可事項
- ファイルの閲覧・参照
- 新規.txtファイル作成（孔明の指示後のみ）
- Git操作（add/commit/push）
- claude.ai（このチャット）
- Claude Code
- その他全てのAIツール
	作成日：2026年4月2日

## マスター原稿の更新方法
孔明がテキストエディタで.txtファイルを直接編集 → 保存 → それが即マスター
**修正履歴の記録は不要。**
