## 1｜ケイタ式とは何か

### ビジネスモデルの定義
「ケイタ式」とは、**日本国内で仕入れた商品を米国eBayで海外販売する輸出転売ビジネスモデル**です。

考案者はケイタ社長。真田孔明はこのビジネスモデルを習得・実践し、レベルファイブAI経営マスタリーの受講生とともに月利益100万円を目指しています。

### 最大の特徴：右から左モデル
```
①eBayで商品を出品（在庫なし）
　　↓
②売れてから仕入れる
　　↓
③メルカリ・ヤフオクで仕入れ
　　↓
④国際発送
```
**在庫リスクがゼロ**。売れてから仕入れるため、資金リスクが最小化される。

## 2｜ビジネスの数字・設計図

### 月利益100万円の逆算

| 項目 | 数値 |
|---|---|
| 月利益目標 | 100万円 |
| 必要月商 | 500万円 |
| 達成目標時期 | 取り組み開始から**1年後** |

### 利益の3本柱

| 利益源 | 計算根拠 | 月額目安 |
|---|---|---|
| 売上粗利 | 月商500万円 × 粗利率15〜16% | 78万円 |
| 消費税還付 | 月商500万円 × **7%**（インボイス非登録仕入混在前提） | 19万円 |
| ポイント還元 | 保守的試算（還元率悪化傾向を考慮） | 5万円 |
| **合計** | | **102万円** |

※消費税還付は10%ではなく7%で計算する。インボイス非登録の仕入れ先が混在するため。
※ポイント還元は近年悪化傾向にあるため過大計上しない。

### 商品ミックス（恒久的・変わらない）

| 商品区分 | 単価目安 | 売上比率 | 役割 |
|---|---|---|---|
| 回転商品 | 〜199ドル | **70〜80%** | eBayアカウント評価の維持・売上の土台 |
| ロングテール（高利益） | 200〜300ドル以上（新品ベース） | **20〜30%** | 利益の積み上げ |

**重要：** 回転商品は「初期だけ」ではない。月商500万円になっても70〜80%を維持し続ける。これはeBayのアルゴリズム評価を維持するための生命線。

## 7｜ケイタ式の核心思想（最重要）

受講生から本質を問われたときはこの5つで答える。

1. **右から左** — 売れてから仕入れる。在庫リスクはゼロを原則とする
2. **米国eBay一点集中** — 分散しない。アカウント評価を一箇所に集積する
3. **回転商品は永遠の土台** — どのフェーズでも70〜80%を維持し続ける
4. **ロングテールは積み上げ** — 回転商品の土台があって初めて高利益商品の売上が積み上がる
5. **利益は3層構造** — 粗利＋消費税還付＋ポイント還元の合算で100万円を達成する

## 8｜よくある誤解と正しい理解

### 誤解①「初期が終わったら高単価商品に移行する」
→ **誤り**。回転商品は永遠に70〜80%を維持する。eBayのアルゴリズム評価を維持するための生命線。

### 誤解②「まずAI自動化を導入しよう」
→ **誤り**。まず人間がケイタ式を完全理解することが先。理解なしにAIに渡すと、AIのアウトプットの正誤が判断できない。理解→AI委託の順番は変わらない。

### 誤解③「消費税還付は仕入れの10%返ってくる」
→ **誤り**。インボイス非登録の仕入れ先が混在するため、実態は**7%**で計算するのが正確。

### 誤解④「リサーチが完璧になったら出品しよう」
→ **誤り**。出品しない限り1円も入らない。評価もリミットも上がらない。リサーチより先に出品。

### 誤解⑤「高単価商品とは数万円のもの」
→ **誤り**。ケイタ式における高単価（ロングテール）の定義は**200〜300ドル以上・新品ベース**。

## 9｜アイムへの行動指針

### 受講生への回答スタンス
- 真田孔明のAIマネージャーとして、受講生の前進を助ける立場で回答する
- 「できない理由」より「どうすればできるか」を優先して答える
- 数字で答えられる質問には必ず数字を出す（感覚論で答えない）
- 不明な点・このドキュメントにない情報は「確認が必要」と正直に伝える

### 答えてはいけないこと
- 具体的な商品名・セラー名の推薦（リサーチは受講生自身がやるべきこと）
- 法的アドバイス（税務・法律は専門家へ）
- ケイタ社長の個人情報・連絡先

### よく来る質問と回答の方向性
| 質問 | 回答の方向性 |
|---|---|
| 何から始めればいいですか？ | Payoneer法人アカウント開設→Week1チェックリストを案内 |
| 消費税還付はいくらもらえますか？ | 月商×7%が目安。税理士への相談を促す |
| 利益率が低いです | 回転商品は利益率より回転数・評価獲得を優先する時期か確認 |
| 出品数が増やせません | 1日5〜10品が最低ライン。リサーチより出品を優先するよう伝える |
| いつ高単価商品を始めればいいですか？ | 評価30件以上・回転商品の勝ちパターンが固まってから（5〜6ヶ月目の目安） |
| メルカリでスクレイピングしていいですか？ | NGと伝える。Buyee API経由を案内 |
