## 株式投資家のための企業研究　　　　　　　　　　　　　　「Amazon（AMZN）」

### 概要
- アメリカ合衆国のワシントン州シアトルに本拠地を置く企業。主軸はインターネット経由の小売であるが、その他にもクラウドコンピューティングなどを手掛けている
	- 2024年2月26日、同日取引分からダウ平均株価の構成銘柄として採用
- CEO
	- アンディ・ジェシー
		- かつてはAmazon Web ServicesのCEOを務めていた
			- ニューヨーク州スカーズデール在住のマージェリーとエベレット・L・ジャシーの間で育ったハンガリー系ユダヤ人
				- 父はニューヨークの法律事務所デューイ・バランタインのロサンゼルス事務所のマネージング・パートナー
				- ハーバード大学で学士号を取得し、ハーバード・ビジネス・スクールでMBAを取得
					- ハーバード大学の中でも優秀な成績。成績優秀者のみに与えられる「Latin Honours」の称号を取得
				- 1997年、Amazonにマーケティングマネージャーとして入社
					- 1997年にカリフォルニア州サンタモニカのロウズホテルにて、シアトルの衣料品小売店エディ・バウアーのファッションデザイナーでフィラデルフィア繊維科学大学の卒業生であるエラナ・ロシェル・キャプランと結婚
				- 共同創業者兼CEOだったジェフ・ベゾスのサポート役を務めたほか、2003年に立ち上げられたAmazon Web Servicesでは57名をまとめるマネージャー職に選ばれた
				- 2016年まではAWSにてSVP、2023年まではCEOをそれぞれ務めた
				- 2021年2月2日、ベゾスが取締役会長に移行することから、後任としてAmazonのCEOに任命されたことが発表
				- 同年7月5日、正式にAmazonのCEOに就任（53歳の時）
					- 決行した大量リストラ
						- 2021年
						- アマゾンは新型コロナのパンデミックの恩恵をたっぷりと受けていた。しかし加速した成長は間もなく壁に突き当たる
							- パンデミック禍での積極投資という「正しい決定をチームは行った」と語る一方で、自社倉庫の拡大に急ブレーキをかけるとともに、バージニア州北部で計画していた第2本社の最大区画の建設に待ったをかけ、2万7000人の従業員を解雇
						- 本格的な成果がまだ出ていないプロジェクトを終了もしくは縮小し、音声アシスタント「アレクサ」やドローン、無人店舗を担当していた従業員をレイオフ
						- 従業員数は昨年初頭ピークの162万人から現在は146万人に減った
					- 「改革」は実を結び始めた
						- 会社の成長は勢いを取り戻し、利益はウォール街の予想を上回って拡大
							- 1年前にはわずかな赤字だったのに対し、大きな黒字化
					- ジャシーの成功は「サービス」分野を強めたこと
						- 人工知能（AI）や医療分野での野心を語り、物流再構築によってより多くの商品をより早く届けるという目標を示している
- 資金の滞留システム
	- Amazonの倉庫に商品を置いているメーカー・卸・小売
		- Amazonからの支払いは2ヶ月後
		- ユーザーからの入金は早い
		- 1ヶ月以上の資金滞留
			- 兆円規模の滞留資金
				- 税金のかからない兆円単位の資金を自由に使える状態
			- 物販は「売上高」を作るのは簡単
				- Amazonの売上高がドンドン伸びていく
					- ＝滞留資金がドンドン増えていく
					- 滞留資金は利益ではないので法人税を払わなくて良い
						- むしろ黒字にならないように投資に回す
					- ミライの成長のための設備投資に充てられる

### クラウド&生成AI大戦
- AWS
	- Amazon社内で2003年からスタートしたプロジェクト
		- 2010年にサービス開始。急速に成長を遂げているクラウドプラットフォーム
			- クラウドコンピューティングプラットフォーム
				- 元々、自社のインフラストラクチャーを効率的に活用するために開発
					- 2006年にサービスを一般提供開始し、徐々に市場規模を拡大
						- 顧客のニーズに合わせたサービスの提供と高品質なカスタマーサポートを重視
			- 競合
				- Microsoft Azure
				- Google Cloud Platform
	- AWSのビジネスモデル
		- 顧客が必要とする機能をクラウド場で提供することで収益を得ている
			- インフラストラクチャー、プラットフォーム、アプリケーション、セキュリティなどの様々なサービスを提供
				- EC2（Elastic Compute Cloud）
					- AWSのクラウドコンピューティングサービスの中核になる製品。仮想マシンを提供するサービス
						- アマゾンマシンイメージ（AMI）を利用して、顧客が必要とするOSやアプリケーションを簡単に展開可能。顧客は必要な時間だけEC2を使用し、使用した分だけ課金される
				- S3（Simple Storage Service）
					- AWSが提供するストレージサービス
						- 顧客は、大容量のデータを安全に保存することができる。オブジェクトストレージとして知られており、顧客は必要に応じて容量を拡大することができる。顧客は使用した分だけ課金される
				- RDS（Relational Database Service）
					- AWSが提供するデータベースサービス
						- MySQL、PostgreSQL、Oracle、Microsoft SQL Serverなどのデータベースを提供
							- データベースのセットアップや管理を簡単に行うことができる。顧客は使用した分だけ課金される
				- Lambda
					- AWSが提供するサーバレスのコンピューティングサービス
						- 顧客はコードを実行するためのサーバーを用意する必要が無い。イベントに対してコードを実行することができる。顧客は実行した時間やメモリの使用料に応じて課金される
				- その他
					- API Gateway、CloudFront、Elastic Beanstalk、CloudFormation、CloudTrail、CloudWatch、DynamoDB、ElastiCache、Kinesis、Redshift、Route53、SES、SNS、SQS、VPCなどがある
				- AWSのビジネスモデルは、顧客が必要とする機能を提供することで収益をえるものであり、多数のサービスを提供することで収益を産み出し続けている
				- リージョンやアイベラビリTゾーン（AZ）と呼ばれる地理的に分散したデータセンターを提供することで、世界中の顧客にサービス提供
- Microsoft
	- Azure

					- 不正アクセスの防止やサイバー攻撃への対策ができる「Azure Security Center」の使用ができるだけでなく、政府機関などと連携してセキュリティー対策の研究拠点を世界に展開し、高いセキュリティーレベルを担保

				- Windows ServerやActive Directoryなど、多くの企業で利用されているMicrosoftの製品やサービスとの親和性が高い

				- 企業は既存のMicrosoft環境をスムーズにクラウドに拡張できる。新たなツールやプロセスを一から習得することなく、クラウドのメリットを享受できる

### Amazonの人事戦略
- OLP（Our Leadership Principles）
	- Amazon150万人の社員を群で動かす武器
	- 16項目から成るリーダーシップ
		- Amazonで働く人は、全てこのOLPに沿って考え、行動し、結果を出している
			- OLPに合致する人を採用し、入社後はOLPを実行できている人を高く評価
				- 不透明な部分が無い
				- 全世界で共通
	- 1）Customer Obsession（顧客へのこだわり）
		- リーダーはお客様を起点に考え行動する。お客様から信頼を獲得し、維持していくために全力を尽くす。競合にも注意を払うが、何よりもお客様を中心に考えることにこだわる
			- 全世界で取り組む「Our mission」
				- 「Earth’s most customer centric company（地球上でもっともおきゃくさまを大事にする会社）」
	- 2）Ownership（オーナーシップ）
		- リーダーにはオーナーシップが必要。長期的視点dね考え、短期的な結果のために、長期的な価値を犠牲にしない。自分のチームだけでなく、会社全体のために行動する。「それは私の仕事ではありません」は決して口にしてはならない
	- 3）Invent and Simplify（創造と単純化）
		- リーダーはチームにイノベーション（核心）とインベンション（創造）を求め、同時に常にシンプルな方法を模索。状況の変化に注意を払い、あらゆる場から新しいアイディアを探し出す。新しいアイディアを実行に移す時、長期間に渡り外部に誤解される可能性があることも受け入れる
	- 4）Are Right, A Lot（多くの場合正しい）
		- リーダーは多くの場合、正しい判断を行う。優れた判断力と経験に裏付けされた直感を備えている。多様な考え方を追求し、自らの考えを反証することもいとわない
	- 5）Learn and Be Curious（学び、そして興味を持つ）
		- リーダーは常に学び、自分自身を向上させ続ける。新たな可能性に好奇心を持ち、探求する
	- 6）Hire and Develop the Best（ベストな人材を確保し育てる）
		- リーダーは全ての採用や昇進における、評価の基準を引き上げる。優れた才能を持つ人材を見極め、組織全体のために積極的に活用する。リーダーが他のリーダーを育成し、コーチングに真剣に取り組む。全ての社員がさらに成長するための新しいメカニズムを創り出す
	- 7）Insist on the Highest Standards（常に高い目標を掲げる）
		- リーダーは常に高い水準を追求することにこだわる。継続的に求める水準を引き上げ、チームがより品質の高い消費にゃサービス、プロセスを実行できるように推進。水準を満たさないものは実行せず、問題が起こった際には確実に解決し、再び同じ問題が起きないように改善策を講じる
	- 8）Think Big（広い視野で考える）
		- 狭い視野で施工すると、大きな結果を得ることはできない。リーダーは大胆な方針と方向性を示すことによって結果をだす。お客様のために従来と異なる新しい視点を持ち、あらゆる可能性を模索する
	- 9）Bias for Action（とにかく行動する）
		- ビジネスではスピードが重要。多くの意思決定や行動はやり直すことができるので、大掛かりな検討を必要しない。計算した上でリスクを取ることに価値がある
	- 10）Frugality（質素倹約）
		- より少ないリソースでより多くのことを実現する。倹約の精神は、創意工夫、自立心、発明を育む源となる。スタッフの人数、予算、固定費は抑える
	- 11）Earn Trust（人々から信頼を得る）
		- リーダーは注意深く耳を方向け、率直に話し、相手に対して経緯を持って接する。間違いは素直に認め、自分やチームの間違いを正当化しない。常に自らを最高水準と比較して評価する
	- 12）Dive Deep（より深く考える）
