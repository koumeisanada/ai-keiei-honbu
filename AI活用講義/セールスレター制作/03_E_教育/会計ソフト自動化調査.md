## 1. マネーフォワード（MoneyForward）関連ツール

### 1-1. sammrai/mfapi — MoneyForward API Proxy
- **URL**: https://github.com/sammrai/mfapi
- **言語**: Python
- **機能**: マネーフォワードの非公式APIプロキシ。公式APIが限定公開のため、プロキシ経由でデータ取得を可能にする
- **用途**: 入出金データの取得、資産情報の参照
- **Claude Code連携**: Python経由で呼び出し可能。スクレイピング型のため認証管理が必要

### 1-2. hiroppy/mf-dashboard — マネーフォワードMe自動化＆資産可視化
- **URL**: https://github.com/hiroppy/mf-dashboard
- **言語**: TypeScript（Next.js）
- **機能**: マネーフォワードMeのデータを自動取得し、保有資産をダッシュボードで可視化
- **仕組み**: Playwright（ブラウザ自動操作）で毎日データ取得 → SQLite保存 → Cloudflare Pagesで公開
- **特徴**: GitHub Actions + 1Password連携で完全自動化
- **Claude Code連携**: 直接連携は困難（ブラウザ自動操作が前提）。ただしSQLiteデータの分析は可能

### 1-3. MH4GF/scraping_moneyforward — MoneyForward MEスクレイピング
- **URL**: https://github.com/MH4GF/scraping_moneyforward
- **言語**: Ruby
- **機能**: MoneyForward MEをスクレイピングし、今月の収支をSlackに自動送信
- **特徴**: Docker + ChromeDriverで動作
- **Claude Code連携**: スクレイピング結果のデータ処理・分析は可能

### 1-4. wywy-llc/mf-invoice-api — マネーフォワード クラウド請求書API GASクライアント
- **URL**: https://github.com/wywy-llc/mf-invoice-api
- **言語**: Google Apps Script (GAS)
- **機能**: マネーフォワード クラウド請求書API v3のGASクライアントライブラリ
- **用途**: Google Spreadsheetからクラウド請求書の作成・管理を自動化
- **Claude Code連携**: GAS環境が必要。Claude Codeから直接は不可だが、スクリプト生成の支援は可能

### 1-5. takanory/anapay2moneyforward — ANA Pay → MoneyForward連携
- **URL**: https://github.com/takanory/anapay2moneyforward
- **言語**: Python
- **機能**: ANA Payの決済メールからデータ抽出し、マネーフォワードに自動登録
- **Claude Code連携**: Pythonスクリプトの実行・カスタマイズが可能

### 1-6. kempei/mfstockupdater — 株価自動更新ツール
- **URL**: https://github.com/kempei/mfstockupdater
- **言語**: Python
- **機能**: Seleniumを使ってマネーフォワードの株価情報を自動更新
- **Claude Code連携**: Selenium環境が必要

### 1-7. moneyforward/api-doc — 公式APIドキュメント
- **URL**: https://github.com/moneyforward/api-doc
- **注意**: 現在はクローズドAPI（特定パートナー企業のみ利用可能）

### 1-8. moneyforward/expense-api-doc — クラウド経費API
- **URL**: https://github.com/moneyforward/expense-api-doc
- **機能**: クラウド経費・クラウド債務支払のAPIドキュメント
- **認証**: Client ID / Client Secretによる認証が必要

## 2. freee（フリー）関連ツール

### 2-1. ⭐ freee/freee-mcp — freee公式MCPサーバー【最重要】
- **URL**: https://github.com/freee/freee-mcp
- **言語**: TypeScript
- **機能**: Model Context Protocol（MCP）サーバーとしてfreee APIをClaude Desktop/Claude Codeから直接利用可能
- **対応API**: 会計・人事労務・請求書・勤怠・販売の5つのfreee API
- **特徴**:
  - OAuth 2.0 + PKCE認証（安全な認証フロー）
  - 自動トークンリフレッシュ
  - 複数事業所対応
  - Claude Plugin機能（API仕様書をスキルとして提供）
- **Claude Code連携**: **直接連携可能** — MCPサーバーとして設定するだけでClaude Codeからfreee操作が可能
- **できること**: 経費登録、取引管理、請求書作成、仕訳入力、勤怠管理など

### 2-2. RyutoYoda/freee-chat-mcp-server — freee会計チャットMCPサーバー
- **URL**: https://github.com/RyutoYoda/freee-chat-mcp-server
- **言語**: Python
- **機能**: freee会計APIをClaude Desktopで使えるMCPサーバー
- **特徴**:
  - レシート画像アップロード対応
  - スマート経費登録機能
  - 過去の取引を参照して正確なデータ入力
- **Claude Code連携**: **直接連携可能** — MCPサーバーとして利用

### 2-3. 0xkohe/freee-hr-mcp — freee人事労務MCPサーバー
- **URL**: https://github.com/0xkohe/freee-hr-mcp
- **機能**: freee人事労務APIのMCPサーバー
- **用途**: 従業員一覧取得、勤怠管理、有給申請など
- **Claude Code連携**: **直接連携可能**

### 2-4. datagrid-okada/python-freee-api — Python freee APIクライアント
- **URL**: https://github.com/datagrid-okada/python-freee-api
- **言語**: Python
- **機能**: 会計freee・人事労務freee用のPython APIクライアントライブラリ
- **対応操作**: 経費申請、請求書作成・取得・更新、品目管理、仕訳操作、取引先管理
- **認証**: OAuth認証（初回は認可コードが必要、以降はリフレッシュトークンで自動更新）
- **Claude Code連携**: Pythonスクリプトとして実行可能

## 5. 推奨アクション

### 即座に導入可能（最優先）
1. **freee/freee-mcp** をClaude CodeのMCPサーバーに追加
   - freeeの会計・請求書・経費を自然言語で操作可能になる
   - 設定方法: `~/.claude/settings.json` にMCPサーバー設定を追加

### 短期的に導入可能
2. **python-freee-api** をインストールしてPythonスクリプトで自動化
3. **freee-chat-mcp-server** でレシート画像からの経費登録を自動化

### マネーフォワード利用者向け
4. **sammrai/mfapi** でデータ取得の自動化を検討
5. 公式APIがクローズドのため、スクレイピング型ツールが中心となる点に注意

### 弥生会計利用者向け
6. **arikawa/Siwake** を活用してCSVインポートデータの自動生成
7. Claude Codeで弥生会計フォーマットのCSV生成スクリプトを作成

---

## 6. 特記事項

- **freeeが最もAI連携に積極的**: 公式MCPサーバーを提供しており、Claude Code/Claude Desktopとの親和性が最も高い
- **マネーフォワードはAPIが限定的**: 公式APIはパートナー企業向けクローズドAPI。非公式ツールはスクレイピングが中心
- **弥生会計はAPI未公開**: CSV連携が唯一の自動化手段。ツール数も限定的
- **セキュリティ注意**: OAuth認証情報・APIキーの管理には十分注意すること
