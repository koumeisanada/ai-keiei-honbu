## 受講生への配布方法

### 方法A：PDFに変換して配布（推奨）
```bash
# MacのターミナルでMarkdown→PDF変換
brew install pandoc
pandoc 真田孔明_セールスレタースキル_受講生配布版.md -o セールスレタースキル.pdf
pandoc 真田孔明_文章執筆スキル_受講生配布版.md -o 文章執筆スキル.pdf
```
作成したPDFをGoogleドライブにアップロード→共有リンクをDiscordで配布。

### 方法B：Notionページで配布（最も見やすい）
1. Notionで新規ページを作成
2. Markdownの内容をそのままコピペ（NotionはMarkdown対応）
3. ページを「Webで公開」に設定
4. URLをDiscordやメルマガでシェア

### 方法C：GitHubのRAWリンクで配布
1. GitHubの各ファイルを開く→「Raw」ボタンをクリック
2. そのURLをDiscordでシェア
3. 受講生はURLを開いてコピーまたは右クリックで保存
