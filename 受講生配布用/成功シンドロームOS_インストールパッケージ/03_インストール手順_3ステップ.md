# 成功シンドロームOS インストール手順（3分で完了）

## ステップ1：AI本部Chatを開く（30秒）
あなたが普段使っている「AI本部」のChatを開いてください。

## ステップ2：指示文をコピペ（1分）
1. 01_AI本部への指示文_コピペ用.txt を開く
2. 全文をコピー（Ctrl+A → Ctrl+C）
3. AI本部Chatの「Custom instructions」または「プロンプト設定」に貼り付け

貼り付け場所：
- Claude.ai：プロジェクト設定 → Custom instructions
- ChatGPT：Settings → Personalization → Custom instructions
- Gemini：カスタムGem設定 → Instructions

## ステップ3：SKILLファイルを保存（1分30秒）
02_SKILL_成功シンドロームOS_完全版.md を以下に保存：
~/Desktop/AI経営本部/スキル/成功シンドロームOS/

## インストール完了確認（30秒）
AI本部Chatに入力：
「成功シンドロームOSの5ステップを使って、年収2,000万円を達成する目標の設計図を作ってください」

正しくインストールされていれば、1.目的設定→2.情報収集→3.分析→4.設計図→5.実行 の順序で回答します。
