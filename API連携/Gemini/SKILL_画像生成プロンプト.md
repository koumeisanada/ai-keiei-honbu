# 画像生成スキル（ナノバナナプロ Style）

## 基本プロンプト（毎回必ず使用）
「ナノバナナプロで霞が関のポンチ絵のように、すべての情報を一枚の画像にまとめて。」

## 使い方
メルマガ原稿・講義資料・セミナー資料が完成したら、
以下のプロンプトをGeminiに貼り付けること：

---
ナノバナナプロで霞が関のポンチ絵のように、すべての情報を一枚の画像にまとめて。

【まとめる内容】
[ここにメルマガ原稿または資料の内容を貼り付ける]
---

## 背景画像の選び方（重要）
- 背景は「本文の内容・テーマ」に合った画像を選ぶこと
- 人物写真に限定しない
- テーマ別の背景例：

米国株・投資関連
→ ウォール街・株価チャート・都市夜景・ビル群・ドル紙幣

銀行融資・法人・ビジネス関連
→ 銀行建物・契約書・握手・オフィス・スーツの人物

物販・ケイタ式関連
→ 段ボール・倉庫・EC画面・パソコン作業・輸出港

成功・資産・FIRE関連
→ 豪邸・高級車・リゾート・海・自由な人物シルエット

AI・テクノロジー関連
→ デジタル回路・データ・未来都市・光の粒子

子育て・教育・ワンチーム関連
→ 親子・学習・笑顔の子供・教室・本

マインド・成功シンドロームOS関連
→ 山頂・夜明け・炎・走る人物・上を向く人物

### 背景選定のルール
1. 本文を読んでテーマを判断する
2. 上記の例から最も近い背景を選ぶ
3. 暗めのトーン・半透明オーバーレイで文字を読みやすくする
4. 背景が主役にならないよう抑えめにする
